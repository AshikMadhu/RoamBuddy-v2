## ✨ About RoamBuddy

**RoamBuddy v2** is a modern **travel planning web application** designed to help travelers organize their trips, explore destinations, and manage travel activities in one place.

It provides a simple and interactive interface that allows users to **plan journeys, track travel schedules, manage bookings, and access emergency support tools** easily.

## 🚀 Key Features

🌍 **Trip Planner**  
Create and organize travel plans quickly and efficiently.

📅 **Daily Travel Schedule**  
Manage and track daily activities during your trip.

🚆 **Train / PNR Status Tracking**  
Check train information and travel updates.

🏨 **Booking Management**  
Keep track of travel bookings and details.

🗺️ **Explore Destinations**  
Discover interesting travel locations and plan visits.

🚨 **Emergency SOS Assistance**  
Access quick help and emergency contact features when needed.

📱 **Responsive Design**  
Works smoothly on **desktop, tablet, and mobile devices**.

⚡ **Fast & Lightweight Web App**  
Simple design with smooth user experience.
